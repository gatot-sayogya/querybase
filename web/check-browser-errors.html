How to Check Browser Errors

  1. Open the page where you see errors
  2. Press F12 or Cmd+Option+I (Mac) / Ctrl+Shift+I (Windows/Linux)
  3. Click the "Console" tab
  4. Look for red error messages
  5. Copy those error messages and paste them here

Common Errors to Look For:

  • Failed to fetch
  • Unexpected token < in JSON
  • Cannot read properties of undefined
  • Network request failed
  • Any 404 or 500 errors in Network tab
